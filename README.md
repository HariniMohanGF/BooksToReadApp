# BooksToReadApp

A simple To-Read application built using **Spring Boot** and **MySQL**..


##  Features

Add a new book details to read
Search by ID
Search by book name
View all books
View unread books
Update book details or fields
Delete a book record

---

## URL endpoints
Find the endpoint urls in the POSTMAN collection below.

## Tech Stack

- **Backend:** Spring Boot  
- **Database:** MySQL  
- **Build Tool:** Maven  
- **API Testing:** Postman
